Online filters

  • Those online filters are for display purpose only: the current filter settings are not used anywhere else than in the functions that paint the graphic figures in Brainstorm.
  • Changing the options in this tab does not change the contents of the files in the database.
  • The filters are applied only to the portion of data that are currently displayed, hence give approximate results for long recordings.
  • To filter properly the data and save the results permanently, please use the "Process1" tab.
